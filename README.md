# FoodDeliveryApp

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 13.3.3.

## Installation
 1.Clone repo
   ```sh
   https://github.com/Riopradheep007/Food-App-UI.git
   ```
 2.Install NPM packages
  ```sh
  npm install
  ```
 3.Run
  ```sh
  ng serve
  ```
 4.Application Url
  ```sh
  http://localhost:4200/
  ```
